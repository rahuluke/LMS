LMS Web Application

Live Demo: https://lms-o0og.onrender.com

Overview

This is a Learning Management System (LMS) web application built as part of my Full Stack Development assignment. The project demonstrates my understanding of web development, including frontend and backend integration.

As a beginner, I focused on implementing the core functionalities, though a few advanced features were not completed. This project reflects my learning journey and enthusiasm to improve further.

Features Implemented

User registration and login

Dashboard to view available courses

Basic CRUD operations for courses

Responsive UI

Features Planned (Not Implemented)

Advanced course management features

User role-based access control (Admin/Student)

Technologies Used - 

Frontend: React Tailwind JavaScript

Backend: Node.js, Express.js

Database: MongoDB , clodinary

Deployment: Render
